# Drivalyze - Car Price Prediction App Screenshots

This directory contains screenshots of the Drivalyze application, a car price prediction platform.

## Screenshots

1. **Landing Page**
   - The welcoming page of Drivalyze that introduces users to the car price prediction service.
   - Features a clean, modern UI with a call-to-action to get started.
   - File: `landing-page.png`

2. **Login Page**
   - Secure authentication page for users to log in to their accounts.
   - Includes options for email/password login and social authentication.
   - File: `login-page.png`

3. **Dashboard**
   - User dashboard showing saved predictions and account information.
   - Displays recent activity and quick access to common actions.
   - File: `dashboard.png`

4. **Prediction Page**
   - Interactive form for users to input car details and get price predictions.
   - Shows prediction results and confidence scores.
   - File: `prediction-page.png`

5. **Prediction Results**
   - Detailed view of a price prediction with visualizations.
   - Includes price breakdown and market comparison.
   - File: `prediction-results.png`

## How to Use

1. Navigate to the application at [Drivalyze](http://localhost:3006)
2. Create an account or log in
3. Use the prediction tool to estimate car prices
4. View your prediction history in the dashboard

## Technologies Used

- Frontend: React.js
- Backend: Python/Flask
- Machine Learning: Scikit-learn
- Styling: CSS/React Components
